實用資訊

住宿・交通・出入境
住宿 中洲川端東方快捷酒店

🚇 距離「中洲川端站」約3-4分鐘

✈️ 福岡機場搭地鐵直達（不用轉車！）

Google Map
交通 機場到市區
  • 國際線航廈 → 免費接駁巴士（1號乘車處，每5-10分一班）→ 國內線地鐵站
  • 地鐵空港線到博多 約5分鐘，到天神 約10-15分鐘
  • 建議準備 ICOCA/Suica 或 Apple Pay 交通卡
  • 地鐵一日券 約 ¥640
入境 日本入境流程
  • 持中華民國護照免簽
  • 出發前在 Visit Japan Web 填寫資料，截圖 QR Code
  • 抵達→自助通關KIOSK（掃QR、護照、指紋、拍照）→入國審查→領行李→電子閘口→入境
出境 福岡機場出境流程
  • 3F 報到櫃台（建議提早2.5-3小時）
  • 2F 安全檢查（液體<100ml）
  • 2F 出境審查（撕免稅單據交海關）
  • 免稅店購物 → 登機門
提醒 重要注意
  • 天氣早晚溫差大：薄外套+針織衫+短袖
  • Tax-Free 退稅需帶護照
  • PayPay 普及，信用卡也方便
  • 必備：個人藥物、濕紙巾、行動電源、雨傘
  • eSIM 推薦，免換卡
